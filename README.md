# 🚀 DevInsight AI

An AI-powered GitHub Developer Analytics Platform built using Flask and GitHub API.

## ✨ Features

- 🔍 Search any GitHub user profile
- 👤 Display developer information
- 📂 Analyze repositories
- 🏆 Find top repository
- ⭐ Calculate total repository stars

## 🛠️ Tech Stack

- Python
- Flask
- GitHub REST API
- HTML
- CSS

## ⚙️ How It Works

User enters a GitHub username.

↓

Flask sends request to GitHub API.

↓

Python processes developer data.

↓

Insights are displayed on a web dashboard.
